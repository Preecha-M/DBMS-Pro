## 🚀 Description

This project is a **Frontend Web Application for a Point of Sale (POS) system** for small coffee shops.  
It provides interfaces for order management, menu browsing, dashboards, and user workflows that connect to the backend API.

The app is built with **React + Vite**, designed for responsive UI development and maintainable frontend structure for both academic and practical use.

---

## 🧩 Features

- 🧾 POS order and checkout user interface
- 📋 Menu and category browsing screens
- 🌍 Multi-language support (i18next)
- 📊 Data visualization with charts (Recharts)
- 🔌 Realtime-ready frontend integration (Socket.IO client)
- 🧪 Unit/UI testing setup with Vitest and Testing Library

---

## 🛠 Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Language:** JavaScript (ES Modules)
- **Routing:** React Router
- **HTTP Client:** Axios
- **Internationalization:** i18next + react-i18next
- **Charts:** Recharts
- **Realtime:** Socket.IO Client
- **Testing:** Vitest, Testing Library, Playwright

## ⚙️ Project Setup

```bash
cd client
npm install
```

Create or update your `.env` file in `client/` with the required values.

---

## ▶️ Running the Application

```bash
cd client

# development
npm run dev

# production build
npm run build

# preview production build
npm run preview
```

Frontend default URL:
```
http://localhost:5173
```
